    递减元素使数组呈锯齿状

    给你一个整数数组 nums，每次 操作 会从中选择一个元素并 将该元素的值减少 1 。

    如果符合下列情况之一，则数组 A 就是 锯齿数组 ：

    • 每个偶数索引对应的元素都大于相邻的元素，即 A[0] > A[1] < A[2] > A[3] < A[4] > ...
    • 或者，每个奇数索引对应的元素都大于相邻的元素，即 A[0] < A[1] > A[2] < A[3] > A[4] < ...

    返回将数组 nums 转换为锯齿数组所需的最小操作次数。

    示例 1： 

    1
2
3
    输入：nums = [1,2,3]  
输出：2  
解释：我们可以把 2 递减到 0，或把 3 递减到 1。

    示例 2： 

    1
2
    输入：nums = [9,6,1,6,2]  
输出：4

    提示：
    1 <= nums.length <= 1000
    1 <= nums[i] <= 1000

    思路
    1. 只有减少操作意味着什么？ 如果你想把\(x<=y\) 改成 \(x>y\) 的话，减少 \(x\) 是毫无意义的，只能减少 \(y\)。
    2. 以第一种为例，要想转换成 A[0] > A[1] < A[2] > A[3] < A[4] > ...，根据提示 1，哪些数一定不需要减少？ 偶数位 nums[0],nums[2],⋯ 是一定不需要的。
    3. 分类讨论两类锯齿数组

    class Solution:
    def movesToMakeZigzag(self, nums: List[int]) -> int:
        inf=0xffffffff
        res1=res2=0
        for i in range(len(nums)):
            if i%2==0:
                t=nums[i]-min(nums[i-1] if i-1>=0 else inf,nums[i+1] if i+1<len(nums) else inf)+1
                if t>0:
                    res1+=t
            else:
                t=nums[i]-min(nums[i-1] if i-1>=0 else inf,nums[i+1] if i+1<len(nums) else inf)+1
                if t>0:
                    res2+=t
        return min(res1,res2)

    网络延迟时间

    有 n 个网络节点，标记为 1 到 n。

    给你一个列表 times，表示信号经过 有向 边的传递时间。 times[i] = (ui, vi, wi)，其中 ui 是源节点，vi 是目标节点， wi 是一个信号从源节点传递到目标节点的时间。

    现在，从某个节点 K 发出一个信号。需要多久才能使所有节点都收到信号？如果不能使所有节点收到信号，返回 -1 。

    示例 1：

    1
2
    输入：times = [[2,1,1],[2,3,1],[3,4,1]], n = 4, k = 2
输出：2

    示例 2：

    1
2
    输入：times = [[1,2,1]], n = 2, k = 1
输出：1

    示例 3：

    1
2
    输入：times = [[1,2,1]], n = 2, k = 2
输出：-1

    提示：
    1 <= k <= n <= 100
    1 <= times.length <= 6000
    times[i].length == 3
    1 <= ui, vi <= n
    ui != vi
    0 <= wi <= 100
    所有 (ui, vi) 对都 互不相同（即，不含重复边）

    思路

    Dijkstra 算法

    把所有结点分成两类：

    1. 已确定从起点到终点的最短路径的结点
    2. 未确定从起点到终点的最短路径的结点

    每次从第二类结点中选取一个离起点最近的结点，用它来更新其他第二类结点的距离，然后把它添加到第一类结点中，直到第二类结点为空。

      class Solution:
    def networkDelayTime(self, times: List[List[int]], n: int, k: int) -> int:
        map=[[] for _ in range(n+1)]
        for i,j,v in times:
            map[i].append([j,v])

        minDist=[0xffffffff]*(n+1)
        minDist[k]=0
        visit=set() # 第一类结点
        while len(visit)!=n:
            x = -1
            # 寻找一个离起点最近的第二类结点
            for i in range(1,n+1):
                if i not in visit:
                    if x == -1 or minDist[i]<minDist[x]:
                        x = i
            # 添加为第一类结点
            visit.add(x)
            # 更新其他相连接的第二类结点结点的最短路径
            for i,j in map[x]:
                minDist[i]=min(minDist[i],minDist[x]+j)
        res=max(minDist[1:])
        return res if res!=0xffffffff else -1

      class Solution:
    def networkDelayTime(self, times: List[List[int]], n: int, k: int) -> int:
        map=[[] for _ in range(n+1)]
        for i,j,v in times:
            map[i].append([j,v])
        minDist=[0xffffffff]*(n+1)
        minDist[k]=0
        visit=set() # 第一类结点
        heap=[[0,k]]
        while heap:
            # 选取一个离起点最近的结点

            # 需要理解这一步（比如有一个结点i被重复加入堆，但是距离不一样，这里剔除掉距离不为最短的项） t_dist,x=heapq.heappop(heap)
            if t_dist>minDist[x]: 
                continue
            # 添加为第一类结点
            visit.add(x)
            # 更新其他相连接的第二类结点结点的最短路径
            for i,j in map[x]:
                if minDist[i]>minDist[x]+j:
                    minDist[i]=minDist[x]+j
                    heapq.heappush(heap,[minDist[i],i])
        res=max(minDist[1:])
        return res if res!=0xffffffff else -1

    使用机器人打印字典序最小的字符串

    给你一个字符串 s 和一个机器人，机器人当前有一个空字符串 t 。执行以下操作之一，直到 s 和 t 都变成空字符串：

    • 删除字符串 s 的 第一个 字符，并将该字符给机器人。机器人把这个字符添加到 t 的尾部。

    class Solution:
    def robotWithString(self, s: str) -> str:
        x=0
        bucket=[0]*26 # 字典序 某个字符后的
        for i in range(len(s)):
            bucket[ord(s[i])-ord('a')]+=1
        stack=deque()

        res=''
        i=0
        while i<len(s):
            while not bucket[x]:
                x+=1

            if stack and stack[-1]<=chr(ord('a')+x): # 如果栈顶的字符<=后面所有字符的字典序
                res+=stack.pop()
            else:# 对于每一个字符都需要先入栈
                stack.append(s[i])
                bucket[ord(s[i])-ord('a')]-=1
                i+=1

        while stack:
            rear=stack.pop()
            res+=rear
        return res
